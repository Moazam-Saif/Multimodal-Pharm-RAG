# Pill RAG

Multimodal RAG pipeline for pharmaceutical pill identification from a photo.
Combines computer vision (segmentation + visual embeddings) with text-based
RAG (hybrid retrieval + reranking).

## Status

🚧 Under active development — see `notebooks/` for Colab batch jobs and
`src/pillrag/` for the core pipeline package.
